# QA Automation Framework

Modern Selenium Automation Framework built with Python & Pytest.

## Tech Stack

- Python 3.13
- Selenium 4
- Pytest
- uv
- Ruff
- GitHub Actions
- Loguru

## Features

✅ Page Object Model

✅ Explicit Waits

✅ HTML Reports

✅ Automatic Failure Screenshots

✅ GitHub Actions CI

✅ Parameterized Tests

## Run

uv sync

uv run pytest
